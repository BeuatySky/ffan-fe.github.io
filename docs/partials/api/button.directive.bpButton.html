bpButton
directive in module button

Description

按钮组件, 实际上是对HTML Button的包装, 自定义了一些状态等 包含的状态有

  • enable
  • disabled
  • loading

Usage

as element:
<bp-button
       type="{String}"
       size="{String}"
       html-type="{String}"
       disabled
       icon="{String}"
       click="{ANY}">
</bp-button>

Parameters

ParamTypeDetails
typeString
  • binding symbol is @, 此类型针对的是样式, 比如选了primary class会出现 "bp-btn-primary"", 默认值是primary, 可选值["default", "primary", "danger"]
sizeString
  • binding symbol is @, 按钮大小, 和bootstrap一样, 不填则是默认大小
htmlTypeString
  • binding symbol is @, 顾名思义, 代表的是原生html里的type值
disabledBoolean
  • binding symbol is <, 禁用状态
iconString
  • binding symbol is @, 按钮文字前方的icon名称, 详细见bootstrap
clickANY
  • binding symbol is &, click处理器的表达式
