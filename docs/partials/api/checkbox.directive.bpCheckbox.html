bpCheckbox
directive in module checkbox

Description

包装了HTML Input[type="checkbox"] 包含的状态有

  • enable
  • disabled
  • halfChecked
  • checked

Usage

as element:
<bp-checkbox
       disabled
       checked
       half-checked="{Boolean}"
       true-value="{ANY}"
       false-value="{ANY}">
</bp-checkbox>

Parameters

ParamTypeDetails
disabledBoolean
  • binding symbol is <, 禁用状态
checkedBoolean
  • binding symbol is =?, 是否被选中状态
halfCheckedBoolean
  • binding symbol is =?, 是否是半选状态
trueValueANY
  • binding symbol is @, like ng-true-value, 是对ng-true-value的封装
falseValueANY
  • binding symbol is @, like ng-false-value, 是对ng-false-value的封装
