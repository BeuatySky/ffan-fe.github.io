bpRadio
directive in module radiobox

Description

包装了HTML Input[type="radio"] 包含的状态有

  • enable
  • disabled
  • checked

Usage

as element:
<bp-radio
       disabled
       disabled
       model="{String}"
       list="{Array.Object}"
       complex="{ANY}">
</bp-radio>

Parameters

ParamTypeDetails
disabledBoolean
  • binding symbol is <, 禁用状态
disabledBoolean
  • binding symbol is <, 是否横向展示
modelString
  • binding symbol is =, 绑定在组件上的model
listArray.Object
  • binding symbol is @, 选中后的值
complexANY
  • binding symbol is =?complex, 因为value只能提供String, 复杂结构数据类型使用, complex指令

Example

<bp-radiobox model="vm.radio" value="1"></bp-radiobox>
<bp-radiobox model="vm.radio" value="2"></bp-radiobox>
