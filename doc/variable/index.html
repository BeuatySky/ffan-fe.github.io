    public ButtonComponentDefine: Object source 

    import ButtonComponentDefine from 'anaheim/src/components/button/button.component.js'

    Properties:

    NameTypeAttributeDescription
    type String

    binding symbol is @, 此类型针对的是样式, 比如选了primary class会出现 bp-btn-primary, 默认值是primary, 可选值[default, primary, danger]
    size String

    binding symbol is @, 按钮大小, 和bootstrap一样, 不填则是默认大小
    htmlType String

    binding symbol is @, 顾名思义, 代表的是原生html里的type值
    disabled Boolean

    binding symbol is <, 禁用状态
    icon String

    binding symbol is @, 按钮文字前方的icon名称, 详细见bootstrap
    click Boolean

    binding symbol is &, click 处理器
